# 话剧舞台版工作流 — 电影思维克服指南

## 📌 问题诊断

当前系统发现的主要问题：
1. ❌ **大纲按节拍组织**：每个节拍被当作一个独立单元，导致"虽然理论上是一幕，但实际分成了多个場景"
2. ❌ **节拍转化为场景**：写作时将每个Beat变成一个新的[时间:...]、[地点:...]，导致演员频繁走场
3. ❌ **电影镜头思维**：每个场景就像换一个镜头，导致场景过多
4. ❌ **演员无法表演**：演员在演出中需要频繁切换地点和重新进入角色状态，破坏了表演的连贯性

---

## ✅ 话剧舞台版的正确做法

### 原则1：一幕 = 一个演员可以连贯完成的表演单元

**错误示范（电影版）**：
```
## 第一幕

### 第一场：办公室
[时间:下午3点]
[地点:主任办公室]
(林晚进门...)
林晚: 我想...
主任: 你...

[灯暗转]

### 第二场：走廊
[时间:下午3点5分]
[地点:办公楼走廊]
(林晚离开办公室...)
林晚: (在走廊里)...
```
❌ 问题：
- 演员从办公室走到走廊 = 演员要走下舞台 or 换场景
- 观众看的是"演员走路"而不是"故事进行"
- 演员表演被中断

**正确示范（话剧版）**：
```
## 第一幕

[时间:下午3点，林晚的办公室]
[地点:市政府十楼，主任办公室]
[舞台布景:办公桌、文件、一部老旧的座机]
[人物状态:林晚坐着，面对空着的主任位置。她的手指在桌面上轻轻敲击]

林晚:
  (转身看向窗户)
  他说会在三点回来。
  
  (停顿，看钟)
  
  已经五分了。

(她站起身，走向门边。打开门，往外看了一眼——但她没有走出去)

林晚:
  (转身回到办公桌后)
  还是等着吧。

(她坐下，重新开始敲击桌面...)
```
✅ 优点：
- 所有动作都在同一个舞台空间内完成
- 演员的状态连贯，不需要中断
- 观众看的是"人物心理的变化"而不是"场景切换"

---

### 原则2：每幕最多2个物理空间

**允许的情况**：如果一幕内有两个不同的空间，那么：
1. 空间转换本身必须是**戏剧内容的一部分**
2. 转换时间不超过1分钟
3. 转换过程演员是在舞台上可见的（不是灯灭后的黑屏调度）

**例子**：
```
第一幕内，演员从办公室走到室外打电话

场景1: 办公室 (演员主要在这里)
→ 演员起身，走向舞台侧边
→ 灯光跟随演员
→ 演员踏入舞台侧的一个椅子 (象征室外)
→ 新的灯光设定建立了"室外"的感觉

场景2: 室外 (但演员一整个过程都在舞台上可见，连贯表演)
→ 演员接电话

→ 决定后折回
→ 回到办公室沙发上坐下

整个过程: 演员未曾离开舞台，表演连贯
```

---

### 原则3：节拍是创作参考，不是舞台分割

**错误做法**：
```
【第一幕场景1】

### Beat 1
[时间:...]
[地点:...]
...

### Beat 2
[时间:...] ← 为什么重复场景头?
[地点:...]
...

### Beat 3
[时间:...]  ← 又重复
[地点:...]
...
```

**正确做法**：
```
【第一幕】(一个连贯的故事单元)

[时间:...]
[地点:...]
[舞台布景:...]
[人物状态:...]

(所有的Beat 1、2、3... 都在这个场景内自然发生，没有分割)

人物A:
  台词  ← Beat 1的动作和对话

人物B:
  (动作)  ← Beat 2的推进
  台词

人物A:
  (新的动作)  ← Beat 3的升级
  反驳的台词

(整个幕的内容像是一段连续的表演，看不出"节拍")

(灯光变暗——幕落)
```

## 📋 修改清单：如何识别和改正电影思维

### 自检问题1：场景头出现的频率
❌ 错误：一幕内有5个以上的[时间:...]和[地点:...]
✅ 正确：一幕内最多2个

### 自检问题2：演员走场的频率
❌ 错误：演员在一幕内需要离开舞台再返回
✅ 正确：演员始终在舞台上表演

### 自检问题3：「空场」（没有演员）的长度
❌ 错误：灯灭超过3秒用于切换场景
✅ 正确：无空场或极短的场景过渡（演员在舞台上完成）

### 自检问题4：节拍与场景的关系
❌ 错误：每个Beat都有新的场景头
✅ 正确：多个Beat在同一个场景内进行

## 🔧 大纲修正工作流

### 第1步：检查大纲中的"幕内场景数"

**错误大纲**：
```
第一幕：
  · 场景1：办公室（6个节拍）
  · 场景2：走廊（2个节拍）
  · 场景3：打卡机旁（1个节拍）
```
→ 3个场景，❌ 违反规范

**修正方案**：
- 删除不必要的场景
- 将所有节拍合并到主要场景（办公室）内
- 如果走廊的内容很重要，考虑在办公室中通过电话或其他方式呈现

**修正后大纲**：
```
第一幕：主任办公室（9个节拍在同一空间内）
  · Beat 1-3：林晚在办公室的开场和第一次冲突
  · Beat 4：她决定去走廊打电话
  · Beat 5-6：（通过舞台侧的灯光变化象征走廊，演员始终可见）
  · Beat 7-9：从走廊返回，做出决定
```

### 第2步：检查大纲中是否明确了"场景转换的戏剧性"

如果一幕内确实有2个空间：

**问题清单**：
- [ ] 从空间1到空间2的转换，是否能在1分钟内完成？
- [ ] 这个转换过程本身是否能推进故事（而不是单纯的"走场"）？
- [ ] 演员在转换中是否保持在观众的视野内（舞台可见）？

如果答案都是"是"，那就可以保留这个转换。

## 📝 改写指南：如何改写电影思维的大纲

### 场景超过2个？执行"合并"

**原始大纲**：
```
第一幕:
  · 场景1：办公室（5节拍）
  · 场景2：走廊（2节拍）
  · 场景3：楼梯间（1节拍）
```

**改写步骤**：
1. 识别最重要的场景 = 办公室
2. 对于走廊的内容，问："这能否在办公室中以其他方式表现？"
   - 电话对话？✅
   - 通过舞台侧灯光变化象征走廊？✅
   - 删除这个场景的内容？看是否真的需要
3. 对于楼梯间，问："这1个节拍是否真的必须在楼梯？还是可以并入办公室后的转场？"

**改写后**：
```
第一幕：主任办公室（及通过灯光象征的走廊）
  · 场景：主办公室（主要舞台空间）
  · 增强：舞台侧区域通过灯光转换象征走廊
  · 所有8个节拍在可见的舞台空间内连贯进行
```

## 🎯 验收标准

一个正确的"话剧版"大纲应该满足：

- [ ] 每幕有且仅有 **1个或2个物理空间**
- [ ] 每个空间内 **所有节拍连贯进行**
- [ ] 如有2个空间，**转换过程本身就是戏剧内容**（演员可见）
- [ ] 无论从开始到结束，**演员始终可以参与表演**（无需长时间走场或等待）
- [ ] 观众不会看到"灯灭—场景调度—灯亮"这样的间隙
- [ ] 读大纲时，能清晰想象一个演员在舞台上的完整表演

---

## 💾 为什么这很重要？

1. **可演出性**：电影可以无限切换场景，话剧不行。过多场景会导致：
   - 舞台调度复杂，演员记不住位置
   - 频繁切换造成表演断层
   - 演员从"在故事中"变成"在走场"

2. **观众体验**：观众来看话剧是看**演员的表演**，不是看技术团队换景

3. **制作成本**：每多一个场景，就多一套布景或复杂的灯光设计

4. **表演深度**：演员能在同一空间内更深入地创造人物，而不是被频繁的场景转换打断

## 快速自检表

每次写完大纲或剧本后，问自己：

| 问题 | 是/否 | 修正方案 |
|------|------|--------|
| 这一幕有3个以上不同的物理空间吗？ | ❌是 | 合并场景，用灯光或其他方式象征 |
| 演员在这一幕中需要"走场"吗？ | ❌是 | 重新设计，保证演员始终在舞台上 |
| 如果有场景转换，转换过程是故事的一部分吗？ | ❌否 | 删除这个转换，或重新整合内容 |
| 一个演员从头到尾能不中断地演完这一幕吗？ | ❌否 | 调整场景布置，确保连贯性 |
| 这些节拍能在没有场景分割的情况下自然流畅地展开吗？ | ❌否 | 增强人物动作的因果性，或删除冗余节拍 |
