# 虚拟导演质检清单 (VIRTUAL_DIRECTOR_REVIEW)

每一场戏生成后，必须经过此清单的严格自我质检。

---

## 1. “去广播剧化”检查
- [ ] **静止状态检查**：如果角色只是在说话，是否有足够的肢体微动作（潜台词）补充？
- [ ] **支点交互**：角色是否至少有三次与舞台支点（道具）的产生互动？
- [ ] **潜台词占比**：是否所有台词都过于直白（On-the-nose）？如果能用一个动作代替一句话，请改用动作。

## 2. 冲突密度检查
- [ ] **全员卷入**：场上的每一个角色（包括沉默的角色）是否都通过眼神、动作或呼吸卷入了冲突？
- [ ] **主动权转移**：主动权是否像乒乓球一样在人物间跳动，而不是单方面演讲？

## 3. 主题与逻辑连贯性
- [ ] **次要主题投射**：这三场戏是否清晰地探讨了预设的次要主题？
- [ ] **钩子检查**：结尾是否留出了足以把观众（或下一场戏）钩住的悬念、情绪或动力？

## 4. 舞台可操作性
- [ ] **物理距离**：角色之间的空间距离（逼近、退缩、绕行）是否在动作提示中有所体现？
- [ ] **上下场冲击**：人物的离开是否带有“摔门”、“决然”或“迟疑”等改变舞台氛围的质感？
