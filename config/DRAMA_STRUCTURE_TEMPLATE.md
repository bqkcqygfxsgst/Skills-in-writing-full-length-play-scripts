# 话剧结构拆解模板 (DRAMA_STRUCTURE_TEMPLATE)

在开始生成任何具体的剧本正文之前，必须先按此模板完成拆解。

---

## 1. 场景基本面 [Scene Basic]
- **场次**：第 X 场
- **因果联系**：上一场发生了什么？本场是如何承接的？
- **主要主题定位**：本场如何体现 [主要主题]？
- **次要主题定位**：本场探索了哪个 [次要主题]？

## 2. 人物任务与冲突矩阵 [Character & Conflict]
| 人物名称 | 最高任务 (这一场他想要什么) | 障碍 (谁或什么阻碍了他) | 与他人的冲突点 |
| :--- | :--- | :--- | :--- |
| 角色 A | ... | ... | 与 B：... / 与 C：... |
| 角色 B | ... | ... | 与 A：... / 与 C：... |

## 3. 权力转移路径 [Power Shift]
- **开场权力归属**：谁处于进攻位置？
- **转折点 (Turning Point)**：哪一句话或哪一个动作导致了主动权易手？
- **结场权力归属**：谁在心理或物理上获胜了？

## 4. 舞台支点与动作流 [Anchors & Action]
- **核心支点 (道具)**：本场最重要的物理道具是什么？
- **动作设计**：主要人物将如何通过使用/破坏/争夺这个支点来外化冲突？

## 5. 出入场任务 [Entry/Exit]
- **谁进入？脚本任务是什么？**
- **谁离开？离开时对场内留守者造成了什么冲击？**
