## 问题描述

### 症状
在最初生成的 `最后一班地铁_完整舞台剧本_DeAI纯净版.md` 中出现了对话框架被破坏的问题：
- 对话指示被完全删除，导致舞台指示与台词之间的连接断裂
- 例：`**李杨**：*(用一种想转移注意力的语调)* 我的家人...` 
- 错误变成：`**李杨**： 我的家人...`（看起来像舞台指示被删除了）

### 根本原因
`simplify_dialogue_directions()` 方法中的规则过于激进：
```python
# ❌ 错误规则（完全删除舞台指示框架）
(r'\*\(用一种.*?的声音\)\*', ''),    # 删除整个 *(用一种X的声音)*
(r'\*\(用.*?的语调\)\*', ''),        # 删除整个 *(用X的语调)* 
(r'\*\(语调.*?\)\*', ''),             # 删除所有语调相关指示
```

这直接导致对话标记被删除，破坏了舞台文本的可读性。

---

## 修复方案

### 规则调整：从"删除"改为"简化"

**新规则**（v1.1）：
```python
# ✅ 改进规则（智能简化，保护框架）
(r'\*\(用一种([^*]*?的语调)\)\*', r'*(\1)*'),    # 简化：删除"一种"
(r'\*\(用一种([^*]*?的声音)\)\*', r'*(\1)*'),    # 简化：删除"一种"
(r'\*\(停顿\)\*\s+\*\(停顿\)\*', '*(停顿)*'),    # 合并重复
```

## 技术细节

### 正则表达式对比

**❌ 错误规则（删除）**：
```python
(r'\*\(用一种.*?的声音\)\*', '')
# 输入：*(用一种柔和的声音)* 我爱你
# 输出：我爱你  ← 舞台指示完全消失！
```

**✅ 改进规则（简化）**：
```python
(r'\*\(用一种([^*]*?的声音)\)\*', r'*(\1)*')
# 输入：*(用一种柔和的声音)* 我爱你
# 输出：*(柔和的声音)* 我爱你  ← 舞台指示保留，冗词删除！
```

### 捕获组 `([^*]*?...)` 说明
- `([^*]*?)` ：捕获非 `*` 的任意字符（贪心度最小）
- 作用：保留"柔和的声音"部分，删除"用一种"部分
- 结果：`r'*(\1)*'` 将捕获内容放回指示框架内
