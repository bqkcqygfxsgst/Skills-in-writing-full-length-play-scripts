# 剧本全局记忆 (MEMORY.md)

## 0. 项目全景 [Project Overview]
- **剧名**：玻璃钟罩下的回声
- **核心冲突**：数字虚拟化生存对物理人际关系的彻底侵蚀与阻断。
- **主要主题**：数字时代的家庭异化与物理真实（Physical Reality）的消亡与救赎。
- **次要主题**：
  - 数据与算计的反噬（父亲的破产危机）。
  - 虚构形象带来的虚无与债务悬河（母亲的滤镜骗局与高利贷）。
  - 逃离现实造成的自我伤害与情感冻结（儿子的VR成瘾与自残）。
  - 传统物理情感的不可量化与遗忘（爷爷的老旧情感坚守）。

## 2. 场景进度与伏笔 [Scene Log & Hooks]
*此处记录各幕推进的关键状态与因果钩子：*

- **第一场（第一幕）：透明的茧房**
  - **核心动作**：处于玻璃罩内的四人通过麦克风沟通，互不看对方，展现现代家庭典型的“各自为战”。
  - **关键悬念/伏笔**：
    1. 林致远极度紧张的“对账单”数据——随时引爆的破产炸弹。
    2. 苏雅为了补光灯效果，强行挪动了公文包的位置。
    3. 林诺手腕衣服边缘不经意掠过的一丝血迹。
    4. 林老用尽全力上紧发条却无法插电播放的留声机。
  - **结尾事件**：全城特大停电，玻璃罩上的呼吸灯由正常蓝光变为刺目的红光警报，随后熄灭。麦克风电源切断，所有人被迫用原生嗓音开始讲话。

## 3. 已知世界设定 [Given Circumstances]
- **时间背景**：不久的未来，一个高度依赖AI、大数据与智能物联系统的雷雨之夜。
- **物理空间**：
  - 【唯一舞台核心点】：客厅正中。
  - 【舞台布景规则】：舞台正中央不可见但明确存在的“玻璃立方体”边界。所有演员在第一幕中在这个边界内活动，宛如展柜中的商品。边缘是象征科技冷酷的LED呼吸灯带。
- **核心支点道具**：
  - 林致远的**平板电脑/公文包**（权力和谎言的载体）。
  - 苏雅的**环形补光灯**（虚荣和面具的象征）。
  - 林诺的**VR头显**（逃避与隔绝的盾牌）。
  - 林老的**老式留声机与黄针把手**（物理记忆与原始人际的见证）。
